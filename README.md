## ğŸš€ Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/anshulchahar/gita.git
cd gita
```

### 2. Firebase Setup

1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Create a new project or use existing one
3. Add an Android app with package name: `com.schepor.gita`
4. Download `google-services.json`
5. Place it in the `app/` directory

### 3. Gemini API Setup

1. Visit [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Create a new API key
3. Add it to `local.properties`:

```properties
GEMINI_API_KEY=your_gemini_api_key_here
```

### 4. Install Dependencies

Open the project in Android Studio and sync Gradle files.

### 5. Run the App

Select a device/emulator and click Run (â–¶ï¸)
