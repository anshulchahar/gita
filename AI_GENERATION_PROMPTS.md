## üé¨ Lottie Animations - Creation Strategy

### Tools for Animations:

**Recommended Workflow:**

1. **Frame Generation:** Midjourney or DALL-E 3
   - Generate 5-8 keyframes per animation
   - Maintain consistency across frames
   - Same style as static SVGs

2. **Animation Creation:** Choose ONE:
   - **LottieFiles Creator** (Web-based, easiest) ‚úÖ RECOMMENDED
     - Upload frames as image sequence
     - Set timing and easing
     - Export as Lottie JSON
     - https://lottiefiles.com/creator
   
   - **Adobe After Effects + Bodymovin** (Professional)
     - Import keyframes as layers
     - Animate in After Effects
     - Export with Bodymovin plugin to Lottie JSON
   
   - **Rive** (Alternative, easier than AE)
     - Import artwork
     - Create animations visually
     - Export as Lottie or Rive format

3. **Optimization:**
   - Use LottieFiles optimizer
   - Target <100KB per file
   - Test on mobile devices

**Animation Format:** `.json` files (Lottie JSON format)

## üöÄ Generation Workflow

### For SVGs (Nana Banana):

1. **Go to:** https://nanabanana.ai/
2. **Paste prompt** from this document
3. **Generate** and review
4. **Adjust** if needed (regenerate with tweaks)
5. **Download** as SVG
6. **Optimize** with SVGO if needed
7. **Rename** following convention: `krishna_neutral.svg`
8. **Place in:** `/app/src/main/res/drawable/`

### For Animations:

1. **Generate Keyframes:**
   - Use Midjourney or DALL-E 3
   - Generate each frame using prompts above
   - Download all frames for one animation

2. **Create Animation:**
   - Go to https://lottiefiles.com/creator
   - Upload frames as image sequence
   - Set timing (use specifications above)
   - Add easing curves
   - Add particle effects if needed (confetti, sparkles, notes)

3. **Export:**
   - Export as Lottie JSON
   - Optimize with LottieFiles optimizer
   - Test file size (<100KB)

4. **Integrate:**
   - Rename: `krishna_eating_butter.json`
   - Place in: `/app/src/main/res/raw/`
